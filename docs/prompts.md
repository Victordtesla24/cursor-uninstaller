# Cursor AI Agent Development Prompts

This document outlines prompts for developing and testing the Cursor AI agent.

## Prompt 1: Initial Setup & Configuration

**Goal:** Configure the agent with basic settings and validate its environment.

**Instructions:**

1.  Read the `package.json` file.
2.  Install all dependencies using `npm install`.
3.  Verify that `jest` is installed correctly.
4.  Create a `test.js` file with a simple Jest test.
5.  Run the test and confirm it passes.

## Prompt 2: Code Refactoring

**Goal:** Refactor a given code snippet to improve readability and maintainability.

**Instructions:**

1.  Analyze the provided code snippet.
2.  Identify areas for improvement (e.g., variable names, function length, comments).
3.  Apply refactoring changes.
4.  Ensure all existing tests still pass.
5.  Explain the changes made and the reasoning behind them.

## Prompt 3: Architectural Diagram

**Goal:** Generate a diagram of the project's architecture.

**Instructions:**

1.  Analyze the project structure.
2.  Identify the core components (e.g., AI controller, language adapters, MCP servers).
3.  Create a Mermaid `graph TD` diagram showing the relationships between these components.
4.  The diagram should be simple and easy to understand.
